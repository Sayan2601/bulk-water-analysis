# Scripts

This directory contains Python scripts for post-processing and analyzing bulk water molecular dynamics (MD) simulations.

## Available Scripts

### RDF Analysis

- **rdf_oo.py**  
  Calculates the oxygen–oxygen (O–O) radial distribution function (RDF) of bulk water.

- **rdf_intermolecular.py**  
  Calculates the intermolecular oxygen–hydrogen (O–H) and hydrogen–oxygen (H–O) radial distribution functions by selecting different reference and neighbor atom groups.
